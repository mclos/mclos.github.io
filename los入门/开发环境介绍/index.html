los开发使用的ide

  1. 是基于eclipse开发的。
  2. 本文只介绍专为los开发的功能。

eclispe的工具栏

  • 图标按钮是获取los设备文件。
  • 图标按钮是将编译好的工程打包或打包文件系统。
  • 图标按钮是下载编译好的文件到los的usb设备。（所有文件重新下载）.
  • 图标按钮是只下载los文件.
  • 图标按钮是下载los文件，以及los设备上不存在的文件。
  • 图标按钮是pc端运行los文件。
  • 图标按钮是将当前项目的los文件转换为数组文件。(第一次转换完成后，请在Dubeg文件夹上右键刷新).
  • 图标按钮是将当前项目注册到pc上.
  • 图标按钮是在当前项目的los路径打开losshell.
  • 图标按钮是设置los.

losshell

  1. losshell是管理los的工具。
  2. 使用前请先输入‘h’或‘help’命令。
  3. 注意 使用losshell前必须先安装eclipse TM插件。

新建lgui界面的文件

  1. 点击[File] -> [New] -> [Other] -> [lgui文件] -> [Next]
  2. 输出文件名字，以及lcd大小的参数，点击Flash
  3. 在刚新建的文件上右键，[Open With] -> [lgui编辑器]
  4. 点击弹出窗口的下方[lgui desige]

打开lgui界面的文件

  1. 在选中文件上右键 [Open With] -> [lgui编辑器]
